Horizontal slider

Value set to:

A standard horizontal slider with one value.

  • It should start with a value of 0 and the slider handle all the way to the left of the track.
  • It should be able to be dragged all the way to the end of the track, at which point it should have a value of 100.
  • The slider handle should be keyboard-focusable. When focused, it should move in increments of 1 when the left and right arrow keys are pressed. It should move to the beginning and end of the track when the HOME and END keys are pressed (respectively).
